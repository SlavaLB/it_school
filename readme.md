# Django Lesson Notifications Project

## Описание
Демонстрационный Django-проект, реализующий модель урока (`Lesson`) и механизм уведомлений студентов о событиях урока.

Особенности:
- При создании урока запускается фоновая задача Celery, которая имитирует отправку уведомления студенту через логирование.
- Уведомления учитывают московскую временную зону.
- Реализован WebSocket для возможного реального уведомления в будущем.
- При старте создается суперпользователь username-admin password-admin123

## Технические решения
- **Django Signals** — для отслеживания событий модели.
- **Celery + Redis** — для фоновых задач.
- **Django Channels** — для WebSocket уведомлений.
- **Docker Compose** — для запуска приложения и зависимостей.
- **Volumes** — для сохранения базы данных и логов при перезапуске.
